Here's everything you need to edit the theoryinpractice.ca website from VS Code on your computer. Takes about 10 minutes to set up once, then editing is easy.

First: Check your email (roti@akinsamni.com) for an invite from GitHub to join the theoryinpractice repository. Click Accept invitation before doing anything below.

Part 1 — One-time setup

1. Install VS Code

Download and install from code.visualstudio.com

2. Install Git

Download and install from git-scm.com/downloads (just click through the defaults).

3. Sign in to GitHub from inside VS Code

  1. Open VS Code
  2. Press Cmd + Shift + P (Mac) or Ctrl + Shift + P (Windows)
  3. Type Sign in to GitHub and press Enter
  4. A browser window opens — log in and click Authorize

Part 2 — Download the website files

  1. In VS Code, press Cmd + Shift + P (or Ctrl + Shift + P)
  2. Type Git: Clone and press Enter
  3. Paste this URL:
https://github.com/johhnycrash/theoryinpractice.git
  1. Pick a folder on your computer to save it into (e.g. Documents)
  2. When VS Code asks "Would you like to open the cloned repository?" — click Open

Part 3 — Editing and publishing changes

This is what you'll do every time you want to update the site.

1Pull the latest version first

Click the Source Control icon on the left sidebar (looks like a branching tree), then click the … menu → Pull. This makes sure you have the newest files before editing.

2Edit the file

Open theory_in_practice3.html (or index.html) in the file list on the left. Make your changes and save with Cmd + S (or Ctrl + S).

3Commit your changes

  1. Click the Source Control icon in the left sidebar
  2. In the message box, type a short description of what you changed (e.g. "Update homepage text")
  3. Click the ✓ Commit button

4Push it live

Click Sync Changes (the button that replaces Commit after committing). That's it — your changes are now on GitHub.

5See it live

Wait about 30 seconds, then visit theoryinpractice.ca — Cloudflare automatically deploys every change you push.

✨ Even easier: just ask your AI assistant

If you're using Claude Code or GitHub Copilot Chat inside VS Code, you can skip all the commit/sync steps. After editing the file, just say:

push my changes

Other phrases that work:

  • "save and push to git"
  • "deploy this"
  • "make it live"

The AI will handle the commit and push for you. Wait 30 seconds, then refresh theoryinpractice.ca to see it live.

Installing Claude Code in VS Code

  1. In VS Code, click the Extensions icon in the left sidebar (four squares)
  2. Search for Claude Code
  3. Click Install on the one by Anthropic
  4. Or install from the terminal with this command:
code --install-extension anthropic.claude-code

After installing, click the Claude icon in the sidebar and sign in with your Anthropic account.

Tip: If something goes wrong or you get confused, just message Jonathan. Don't worry about breaking anything — every change is saved in history and can be undone.
